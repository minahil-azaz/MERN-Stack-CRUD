# MERN Stack CRUD Application (React + Vite)

A full-stack CRUD application built using **React (Vite)** for frontend and **Node.js + Express + MongoDB** for backend.  
This project supports **Create, Read, Update, Delete operations** with image upload and real-time UI updates.

---

## 🚀 Features

- Add new users with profile image upload
- View all users in a table
- Edit existing user data (inline modal form)
- Delete users
- Image upload with validation (max 60KB)
- Real-time updates using SWR
- Responsive UI using Ant Design + Tailwind CSS

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Ant Design
- Axios
- SWR
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Multer (file upload)
- CORS
